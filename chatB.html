维度 2 · 空状态与冷启动
每个"空白"都是一次转化机会
核心问题：新用户看到一堆"0"和空白直接流失。每个空状态需要回答三个问题：这里本该有什么？我现在该做什么？别人在做什么？
❌ 原产品

余额显示 ¥0.00 + 空白页面，无引导、无社会证明、无行动指令。用户不知道该充值还是接单。

✅ 优化后

每个空状态预填充数据 + 平台实时社会证明 + 单一明确CTA + 行为经济学驱动的文案。

目标梯度 默认值 从"这是XX功能"升级为"完成这个动作，你就开始赚钱了"
▼
❌ 功能说明式

"这是任务广场，你可以在这里浏览并接取任务。"

→ 用户知道了但不会行动

✅ 任务驱动式

"点击这个任务，看看赚钱有多简单" → 引导完成时用户已经接了第一个任务

→ 引导结束 = Activation 完成

1
聚光灯→顶部任务
"📋 这是适合你的任务，点击它看看赚钱有多简单"
2
聚光灯→接取按钮
"👆 点这里接取任务，完成即可赚取 ¥3"
3
接取成功触发
"🎉 太好了！新手挑战进度 1/3。开始执行 →"
技术实现
遮罩 rgba(0,0,0,0.7) + box-shadow: 0 0 0 9999px 聚光灯
每个Tab独立标记：wx.setStorageSync('tab_X_guided', true)
「我的」→「设置」中提供"重置引导"入口
引导与"新手挑战 0/3"进度打通——引导完成 = 进度 +1
