说

• Pattern A（chatE/F/G/H）：.card-body.show { display:block }
• Pattern B（chatB/C/D）：.card.open .card-body { display:block }

修
tog / toggle / toggleCeo / toggleAll 全部重写：检测当前开/关状态（从 head.open、body.show、card.open 三个标志中任一为真判定"开"），然后同步设置全部三个标志——两种 CSS 模式自动兼容，而且能修复初始状态不一致的 HTML
